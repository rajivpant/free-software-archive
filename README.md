# Free Software Archive (1994-1997)

Source code from my "Free Software & Tips" page at rajiv.com/work/free/, recovered from the [Internet Archive Wayback Machine](https://web.archive.org/web/19971012115614/http://rajiv.com:80/work/free/).

These are tools and scripts I wrote and shared freely between 1994 and 1997 while working at Philadelphia Newspapers (Knight Ridder). The code is preserved here exactly as it was originally published.

## Perl

| File | Date | Description |
|------|------|-------------|
| `perl/split.pl` | August 1994 | Split large files into smaller pieces for transport over slow connections |
| `perl/gifsize.pl` | October 1994 | Return width and height of a GIF image for HTML img tags |
| `perl/go-to-ad.pl` | December 1995 | Ad click-through logging |
| `perl/rotate_ads.pl` | December 1995 | Rotating ad system for .shtml/.asp pages |
| `perl/netscape-index.cgi` | February 1996 | Custom directory listings for Netscape servers (acknowledged in [Netscape Tech Note 20021](https://web.archive.org/web/19971012115614/http://help.netscape.com/kb/server/960513-113.html)) |
| `perl/Date_Time.pm` | May 1996 | Object-oriented date and time functions for Perl 5 |
| `perl/wserver.cgi` | May 1996 | Detect what web server software a site runs |
| `perl/make-dbm-of-urls.pl` | September 1996 | Case-insensitive URL handling for Unix web servers (with Ranjit Bhatnagar) |
| `perl/not_found.cgi` | September 1996 | Smart 404 error handler that adapts message based on referrer |
| `perl/favorites.pl` | ~1996 | Access IE bookmarks remotely via web server |
| `perl/Text-numbers.pm` | September 1997 | Convert numbers to English words (e.g., 9872334 becomes "nine million, eight hundred seventy-two thousand, three hundred thirty-four") |
| `perl/numbers-tryit.pl` | ~1997 | Demo script for Text::numbers |
| `perl/test_Date_Time.pl` | ~1996 | Test script for Date_Time.pm |
